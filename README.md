## :bulb: Installation :bulb:
### Using Exp CLI
```
$ git clone https://github.com/4s1r0c4t/rnrf-expo.git
$ cd rnrf-expo
$ npm install
$ npm install exp --global
$ exp start
```

### Using Expo XDE

`$ git clone https://github.com/4s1r0c4t/rnrf-expo.git`

* Open the rnrf-expo project on Expo XDE.
* Use the "Share" or "Device" buttons to view the project.
